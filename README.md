## 개발 동기
Java로 게시판을 구현 해보면서 Spring Framework에 대한 사용법과 이해도 확립을 목표로 진행하게 되었습니다.

## Environment
항목 | 개발환경 
:---: | :---: 
OS | Windows10 
Language | Java 1.8 
Tool | Spring Tool Suite
Backend | Spring MVC
Frontend | Bootstrap, Jsp, jQuery
WAS | Tomcat 9
DB | Oracle Database 11g Express
ORM | Mybatis
Library | Lombok, HikariCP
